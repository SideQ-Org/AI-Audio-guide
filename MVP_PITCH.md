# AI Audio Guide

> Автономный аудиогид для обычных прогулок. Идёшь — он **сам** рассказывает про то, что
> вокруг; можно **перебить голосом** и спросить, гид ответит и продолжит экскурсию.
> Полностью на лету, hands-free, **8 языков**, работает в кармане с наушником.

---

## ✅ Что готово

**Ядро / агент-рассказчик**
- Новая логика рассказа: **иерархия город → район → улица → объект** — гид ведёт связный
  монолог о районе и **подтягивает реальные объекты** из найденных, вплетая их в рассказ.
- Генерация объектов вокруг из **реальной OSM** (Overpass): радиус, ранжирование по типу/
  близости/направлению взгляда, адаптивный радиус (не оставляет в тишине), дедуп.
- **Триггеры переключения** настроены и работают корректно (эвристический гейт + бесшовное
  переключение на более интересный объект на лету).
- **Отстройка промпта**: умеет **право / лево / прямо / позади** (когда направление надёжное),
  ищет **реально нетипичные факты**, а не «тут находится…».
- **Только факты, без выдумок**: нет фактов → честно называет или молчит, не сочиняет.
- Значимость объектов SKIP → LOW → MEDIUM → HIGH → **LANDMARK** (для знаковых — премиум-модель).
- **Защиты от циклов и повторов**: дедуп нарраций, лимиты «доливания» одного места, фикс
  зомби-процессов.

**Взаимодействие**
- **Hands-free real-time** — открыл и пошёл, рассказ стартует сам по GPS, без единого тапа.
- **Голосовой barge-in** — перебиваешь голосом, гид отвечает тем же голосом, **с памятью
  контекста**, и **продолжает** тур. (плюс текстовый ввод)
- Корректные ответы на вопросы с учётом места и истории.

**Клиент / UX**
- Полноэкранная **карта** (OSM), живая позиция, **кликабельные пины** (текущее/пройденные),
  плавная камера, компас.
- **Тёмная и светлая тема** (живое переключение).
- **8 языков** (en/ru/es/fr/de/it/pt/zh): UI + озвучка (TTS) + распознавание (STT) + сам рассказ;
  автодетект системного языка. Проверено.
- **Фоновый режим** — телефон в карман, экран заблокирован, гид рассказывает в наушник;
  карточка в шторке с **кнопкой Пауза**.
- Архитектура **forward-compatible под аккаунты** (стабильная сессия → при логине добавится
  user_id, контракт менять не нужно).

**Инженерия / прод**
- **Задеплоено и доступно**: публичный хост, TLS, `wss://`, веб-версия + reverse-proxy.
- **Устойчивость связи на реальной прогулке**: heartbeat в обе стороны, **resume сессии** при
  обрыве (та же экскурсия, не с нуля), watchdog на «мёртвый» сокет, реконнект с джиттером.
- **Экономика**: факты бесплатно из **Wikipedia/Wikidata**, платный веб-поиск только для хвоста;
  **жёсткий cap по $**, лимит коннектов на IP, метрика токенов/стоимости в реале.
- **Провайдер-агностичный LLM**: Anthropic / OpenAI-совместимые / OpenRouter / локальный
  LM Studio — переключаются конфигом; обработан региональный геоблок (на проде DeepSeek).
- **Качество**: 106 офлайн-тестов (регресс-гейт), симуляции-харнесс, E2E-прогон по реальным
  городам, **логирование решений агента** (любая прогулка разбираема по логам).
- **Безопасность MVP-тира**: устойчивый WS-приём, спенд-капы, безопасные id (полный аудит проведён).

## 🏗 Архитектура (10 секунд)
Один **stateful оркестратор-«мозг»** (FSM, вся память сессии) + **stateless LLM-роли**
(Scorer / Narrator / Companion). Роли не знают друг о друге — общаются только через состояние
сессии. → чисто, тестируемо, легко менять модель пороли.

## 📊 Метрики (из реальных логов, контрольная прогулка)
- **15 объектов** озвучено за прогулку, переключение почти на каждом, **0 тишины** на близких
  объектах, **повторы района авто-подавляются**.
- **~$0.5–0.7** за длинную (~75 мин) прогулку (в основном бесплатные вики-факты).
- **0 падений**, связь устойчива на мобильном интернете.
- **8 языков**, **106 тестов** зелёные.

## 🎯 Сознательно отложено (это scope MVP, не недоделки)
- Аккаунты / авторизация / **история прогулок** → последний milestone пред-прода.
- Боевой security-тир (per-user auth, заголовки, lockdown транспорта).
- **Прод-карта** (сейчас публичные CARTO-тайлы — для прототипа ок, не под нагрузку).
- Серверный TTS (сейчас on-device — плюс по латентности и цене).
- iOS-бинарь (проект готов, собирается на Mac).

## ⚠️ Честные ограничения (проговорить самому — обезоруживает критику)
- У объектов без вики/веба рассказ тонкий (гид честно называет, но не сочиняет) — структурное.
- `/ws` пока открыт, бэкстоп — спенд-капы; auth — пред-прод.
- Остаточные мягкие повторы-перефразы есть, но «по ощущениям сильно лучше» (есть лог-метрики).

## 💰 Идея монетизации — Freemium

**Модель:**
- **Free** — модель **DeepSeek** (дёшево) + **рекламные баннеры**. Идея: прогулки окупаются рекламой.
- **Paid (подписка)** — более мощная модель (**Gemini**), **без рекламы**, премиум-качество рассказа.

**Почему это технически почти бесплатно внедрить (сильный аргумент):**
- LLM уже **провайдер-агностичный с роутингом по ролям** → «DeepSeek для free / Gemini для paid» —
  это буквально переключатель модели по тиру пользователя, не переписывание.
- **История прогулок** (отложенный аккаунт-функционал) — естественная фича именно платного тира
  и крючок удержания: бесплатно гуляешь, платно — сохраняешь и пересматриваешь свои экскурсии.

**Что честно проверить — unit-экономика (подготовь это, спросят):**
- Длинная прогулка стоит нам **~$0.5–0.7**. Баннерная реклама на мобиле приносит мало
  (eCPM баннеров ~$0.5–2 за 1000 показов) → **с одной прогулки это центы, рекламой как есть
  ~$0.55 не покрыть.** Это главный риск модели — назови его сам.
- **Рычаги, чтобы free-тир окупался:**
  - снизить себестоимость прогулки (агрессивный кэш фактов, короче контекст, реже вызовы
    нарратора — сейчас ~68 вызовов/прогулку с большим контекстом);
  - **лимиты free** (например, N минут/день или N прогулок/мес) — типовая freemium-механика;
  - **rewarded-видео** вместо/вместе с баннерами (платят кратно больше за просмотр);
  - premium-тир легко покрывает свою себестоимость подпиской.

## ❓ вопросы → ответы
- **Работает на реальной прогулке?** → Да, контрольная: 15 объектов, переключение, 0 тишины, есть логи.
- **Сколько стоит прогулка?** → ~$0.5–0.7 за ~75 мин, факты в основном бесплатные, есть жёсткий cap.
- **Если связь упадёт?** → resume сессии + heartbeat + watchdog; 0 падений за прогулку.
- **Почему DeepSeek?** → провайдер-агностично; на проде из-за геоблока OpenAI/Anthropic; меняется конфигом.
- **Готово к пользователям?** → MVP для валидации опыта; до прода — аккаунты, security, прод-карта (роадмап есть).
- **Что дальше?** → фидбэк с живых прогулок → пред-прод (security) → аккаунты + история → монетизация → запуск.
